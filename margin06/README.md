# margin06 — Pair Trading: BTC LONG + ETH SHORT

## Стратегия

Парная торговля на двух коррелированных инструментах.

Одновременно открываются **два ордера** на одинаковую сумму:
- **LONG** BTC-USDT-SWAP
- **SHORT** ETH-USDT-SWAP

## Логика работы

```
Цикл:
  1. Проверяем позиции по BTC и ETH
  2. Если обе открыты → управляем трейлинг-стопами
  3. Если одна закрылась → ждём закрытия второй
  4. Если обе закрыты → открываем новую пару
```

## TP / SL

| Параметр | Значение |
|---|---|
| TP активация | 2 × ATR от цены входа |
| SL активация | 1 × ATR от цены входа |
| TP callback | `callback_atr_multiplier × ATR / цена` |
| SL callback | `callback_sl_atr_multiplier × ATR / цена` |

ATR считается независимо для BTC и ETH. Ордера закрываются независимо.

## Запуск

```powershell
.\trade.ps1              # dryRun режим
.\trade.ps1 -ForceLive   # боевой режим
.\trade.ps1 -DebugMode   # debug вывод
```

## ⚠️ Важно

- Не коммить `config_60m.json` с реальными ключами
- Если BTC открылся, а ETH нет — закрой BTC вручную
